FILE MODIFICATION GUIDE
================================================================================

To Update the Portfolio:

1. Edit Content:
   → modify src/App.jsx
   → change the 't' object (translation/content)
   → run: npm run build

2. Change Colors:
   → modify projectAccents in src/App.jsx
   → run: npm run build

3. Add Project:
   → add to projects object in src/App.jsx
   → run: npm run build

4. Modify Layout:
   → edit JSX in src/App.jsx
   → run: npm run build

5. Deploy:
   → copy dist/ to web host
   → or run: npm run build && deployment command

================================================================================
TROUBLESHOOTING REFERENCE
================================================================================

Issue: CSS not loading
→ Check dist/assets/ folder exists and is uploaded
→ Verify file paths in dist/index.html

Issue: Icons missing
→ Ensure icons.svg is in dist/ root
→ Check browser console for 404 errors

Issue: Language toggle broken
→ Check browser console (F12) for JS errors
→ Verify JavaScript file loaded successfully

Issue: Animations not smooth
→ Normal on older devices
→ Check for console errors
→ Test in Chrome DevTools device emulation

Issue: Assets 404 errors
→ Verify all files from dist/ were uploaded
→ Check file paths in HTML (relative vs absolute)
→ Ensure web server configured for static files
